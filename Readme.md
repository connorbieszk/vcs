vcs is version control system, it also has some project management functionality

vcs commands:
open - open project
list - list avalible projects
add  - add new prokect to list, if without vcs, add it
make - create new project and cd into folder
get  - make a copy of a project
put  - put new files into project
post - updates old files and adds new files to branch
tree - display a branch tree
branch:
    make   - creates a new branch
    rename - rename active branch
    switch - switches branches
delete     - removes all vcs settings from a project
